# MarkBox 演示视频 · 录屏素材清单

录制前先创建目录：
  presentation/public/videos/

录制要求：
  - 1920×1080 分辨率，16:9
  - 不要音频（后期你配音）
  - 每个片段开头和结尾留 0.5s 停顿
  - 提前准备 15-30 条测试书签（不同分类/标签/状态）
  - DeepSeek API Key 确保可用
  - 浏览器扩展提前 npm run build 加载到 Chrome
