# 01_封面

欢迎来到 MarkBox 项目路演。MarkBox 是一款 AI 驱动的智能书签管理器，它解决的问题很简单却深刻：在信息过载的时代，如何让每一个你收藏的链接，真正变成属于你的知识。

---

# 02_困境

我们先来看一个大家都感同身受的困境。收藏从未停止，但整理从未开始。五百多个链接堆在收藏夹里，想找的永远找不到；看到好文章随手就存，存完就再也想不起来看；更重要的是，这些知识散落在各个角落，彼此孤立，永远无法形成体系。

---

# 03_解决方案

MarkBox 给出的答案非常直接。你只需要粘贴一个链接，剩下的全部交给 AI。从自动提取元数据，到智能生成标签、分类和摘要，再到多个视图帮你发现关联和洞察，整个过程三步闭环：收藏、理解、发现。

---

# 04_技术栈

来看一下 MarkBox 的技术基座。全栈 TypeScript，Next.js 十六和 React 十九点二作为全栈框架，Prisma 七点八加上 SQLite 负责数据持久化，AI 引擎用的是 DeepSeek V4 Flash，前端则是 Tailwind CSS 四、shadcn 和 motion 驱动的动画系统。整个技术栈现代、轻量、高效。

---

# 05_首页总览

这是 MarkBox 的首页仪表板。左侧是分类导航侧边栏，中间是统计卡片和书签概览，顶部可以随时切换十种不同的功能视图。一个页面就能让你对自己的知识库有全面的感知。

---

# 06_添加书签

添加书签的体验经过了精心打磨。粘贴一个 URL，系统会自动抓取页面标题、封面图和元数据，然后 AI 会立刻生成标签、分类建议和中文摘要。你可以在预览卡片上确认或修改这些 AI 生成的内容，然后一键入库。整个过程三秒之内完成。

---

# 07_书签管理

书签管理不只是列个清单那么简单。MarkBox 支持 dnd-kit 驱动的拖拽排序，配合乐观 UI 更新，拖拽反馈即时且流畅。每个书签都有四态生命周期：待读、在读、已读、归档，一键切换。分类和收藏夹支持右键菜单和层级树结构，完全自由地组织你的知识。

---

# 08_富媒体卡片

不同来源的内容应该有不同形态的展示。MarkBox 为四种内容类型设计了四种差异化的卡片：视频内容用横向大卡片展示封面和 UP 主信息，文章是纵向卡片配头图，代码仓库是紧凑卡片加语言色条，普通网页则是简约卡片带 favicon 和描述片段。

---

# 09_AI能力总览

AI 是 MarkBox 的灵魂。三项核心 AI 能力让每一个书签从"一个链接"变成"一份结构化知识"。自动标签，AI 生成两到五个精准中文标签；智能分类，自动建议层级分类路径；AI 摘要，两到三句中文摘要让你无需打开原链接就能快速回顾内容。

---

# 10_自动标签

自动标签功能非常实用。DeepSeek V4 Flash 分析书签内容后自动生成精准的中文标签，用户也可以手动增删改。标签会以彩色徽章的形式展示在卡片上，在仪表板中还有标签词云，让知识分布一目了然。更有标签共现推荐功能，基于 Jaccard 相似度算法，毫秒级就能找到关联标签。

---

# 11_智能分类

智能分类解决的是知识组织的问题。AI 基于内容语义自动推断一个层级路径，比如"技术学习、前端、React"。分类树是自引用的父子关系，可以无限嵌套。侧边栏点击分类就能过滤书签，配合 dnd-kit 拖拽重排，整理知识的效率非常高。

---

# 12_AI摘要

AI 摘要让回顾书签变得极其高效。每篇书签入库时，AI 会自动生成两到三句精炼的中文摘要，卡片内嵌展示，点击可以展开完整文本。摘要文本同时纳入全文搜索索引和 Embedding 向量系统，这让语义搜索和智能推荐的效果显著提升。

---

# 13_十大视图总览

这是 MarkBox 最具特色的设计：同一个书签库，十种视角。网格视图适合快速浏览，画廊视图用瀑布流展示图片型书签，发现页有三种模式帮你探索新内容。还有仪表板做数据统计、时间线看收藏历史、横向对比做多维度分析、学习路径做结构化学习计划、周报看每周阅读变化、知识图谱展示关联网络，以及活动日志记录所有操作。

---

# 14_仪表板统计

仪表板是数据驱动的阅读洞察中心。Recharts 渲染的饼图展示阅读状态分布，柱状图展示内容类型比例，纯 CSS 的标签词云显示标签频率。更有三层智能推荐体系：继续深入当前领域、拓展到新的边界、重温被遗忘的旧藏，分别由标签共现算法和 Embedding 语义精排驱动。

---

# 15_时间线与周报

时间线和周报给你时间维度的洞察。时间线视图按月分组、垂直布局，配合 motion 的交错入场动画，翻看自己的收藏历史非常有仪式感。周报则提供本周和上周的数据对比，包括状态分布、热门标签和增长趋势，让你清楚看到自己的知识消费轨迹。

---

# 16_知识图谱

知识图谱是 MarkBox 最具视觉冲击力的功能。力导向图展示书签、标签和分类之间的关联网络。三层交互从标签总览到节点展开再到详情抽屉，层层深入。还有时间轴播放功能，可以以一倍、两倍或五倍速度回放知识网络的生长过程，并且支持截图导出高清 PNG。

---

# 17_横向对比

横向对比让你同时审视两到五篇书签。DeepSeek V4 Flash 会从五个维度进行深度分析，结果以雷达图和矩阵表格的形式呈现。矩阵中的差异会被红黄标记高亮，AI 还会给出总结评语。对比结果可以导出为 Markdown，方便分享和存档。

---

# 18_学习路径

学习路径把散落的书签编排成结构化的学习计划。你可以创建路径、拖拽编排书签顺序、标注难度和阶段。每个节点还可以添加微型笔记，记录收获和疑问。AI 可以分析路径的合理性、检测知识缺口，甚至帮你生成学习路径的骨架。

---

# 19_发现页与推荐

发现页让好书签自己找到你。三种发现模式：跨领域探索带你跳出舒适区，最新收录让你不遗漏新内容，每日精选推荐值得回顾的旧藏。背后的推荐体系分三层：标签共现做快速关联、Embedding 向量做语义精排、Dashboard 做个性化推荐。

---

# 20_浏览器扩展

浏览器扩展让收藏变得无处不在。在任意网页上右键就能一键保存到 MarkBox。Ctrl+Shift+S 快捷键支持静默收藏，不中断你的浏览。弹窗模式提供完整的收藏流程，包括去重检测和 AI 分类建议。侧边栏模式让你在当前页面就能浏览和搜索书签。

---

# 21_语音搜索

语音搜索让找书签像说话一样自然。基于浏览器原生的 Web Speech API，无需额外权限，原生支持中文。边说边显示识别文字，语音结束后自动填入搜索框。麦克风按钮有脉冲动画指示器，录音状态一目了然。结合语义搜索，说一句"上次那个讲 React hooks 的视频"就能找到对应书签。

---

# 22_分享与协作

MarkBox 支持公开分享和协作。你可以把收藏夹设为公开，生成分享链接发送给朋友；也可以生成单条书签的分享卡片。CORS 跨域中间件让浏览器扩展、本地开发环境和自定义域名都能统一访问。同时，桌面固定侧边栏和移动端抽屉式导航确保了各种设备上的体验。

---

# 23_认证与安全

安全是基础设施。MarkBox 采用 GitHub OAuth 作为主要登录方式，基于 NextAuth 版本五。认证中间件采用双重策略：Session 优先，API Key 回退。API Key 以 mb_ 前缀的九十六字符随机密钥生成，仅首次生成时完整显示一次，存储时使用 SHA-256 哈希，明文永远不会落盘。

---

# 24_管理面板

管理面板为团队和实例管理员提供完整的控制能力。你可以查看所有注册用户、分配管理员和普通用户角色、以管理员视角浏览任意用户的书签数据。还提供数据库统计总览卡片，支持一键导出书签数据为 JSON 格式，方便备份和迁移。

---

# 25_开始使用

MarkBox 完全开源，基于 MIT 协议。克隆仓库、安装依赖、推送数据库、启动开发服务器，就这三步，一行命令就能跑起来。三十六条 API 端点、二十八个组件、十种功能视图，全部在 GitHub 上等你来探索。欢迎大家 Star 和 Contribute。
