# 迭代 01 — 项目概念 & MVP 定义

> 日期: 2026-06-02
> 状态: MVP 范围已确定

---

## 项目定位

**书签管理系统** — Web 课程作业，面向期末答辩。

核心差异点：AI 自动整理归类 + 富媒体预览 + 语义搜索。

对标产品：Raindrop（智能书签）、MyMind（AI 自动标签 + 自然语言搜索）、Hoarder（开源，AI 元数据提取 + 截图预览）。

### 方向一：核心体验（✅ MVP）

13. 浏览器扩展一键收藏（工具栏按钮 + 右键收藏）

### 方向六：AI 深度理解（🎯 答辩加分项）
14. AI 摘要 + 关键要点提取
15. 同一主题横向对比

### 方向七：仪式感 & 留存（🔮 后续规划）
16. 每周书签周报
17. 收藏里程碑
18. 随机漫步

### 方向八：搜索新玩法（🎯 答辩加分项）
19. 语音搜索（Web Speech API）

### 方向九：数据可携带（🔮 后续规划）
20. 一键导入/导出（浏览器书签格式、PDF 报告）

### 方向十：微型社交（🔮 后续规划）
21. 个人笔记/评论（给自己写微型读书笔记）

## 架构决策：分层混合方案

```
┌─────────────────────────────────┐
│  AI 层（亮点 — 答辩核心）         │
│  - 语义搜索                       │
│  - 智能标签建议                    │
│  - 自动摘要生成                    │
├─────────────────────────────────┤
│  规则层（稳定 — 本地执行）         │
│  - URL 元数据提取（OG/JSON-LD）    │
│  - 域名→类型映射（b站→视频）       │
│  - 封面图、标题、描述提取          │
├─────────────────────────────────┤
│  基础层（必修 — CRUD）             │
│  - 增删改查                       │
│  - 用户界面                        │
│  - 数据持久化                      │
└─────────────────────────────────┘
```

设计原则：AI 失败不影响基本功能。

## 技术栈（最终确定）

| 层 | 选型 | 说明 |
|---|---|---|
| **前端框架** | React 19 + Vite | 生态最丰富，Claude 对 React 代码质量最高 |
| **UI 组件** | shadcn/ui + Tailwind CSS v4 | 专业、可定制，配合 frontend-design 技能 |
| **全栈框架** | Next.js 16 (App Router) | 前后端合一，一个 `npm run dev` 全启动 |
| **数据库 ORM** | Prisma | 类型安全，与 Next.js 配合最佳 |
| **数据库** | SQLite (better-sqlite3) | 零配置，本地演示无依赖 |
| **AI API** | DeepSeek V4 Pro (`deepseek-v4-pro`) | OpenAI 兼容接口，1M context，49B 激活参数 |
| **AI 接入方式** | `openai` SDK → `base_url: https://api.deepseek.com` | 标准 OpenAI SDK 即可调用 |
| **浏览器扩展** | Chrome Extension Manifest V3 | 二期选做 |
| **部署** | Vercel (免费) | 答辩时线上可访问 |

### DeepSeek API 关键参数
- **Model ID**: `deepseek-v4-pro`
- **Base URL**: `https://api.deepseek.com`
- **接口**: OpenAI 兼容 (`/v1/chat/completions`)
- **Thinking mode**: `non-thinking` / `thinking` / `thinking_max`
- **定价**: $1.74/M input, $3.48/M output
- **Context**: 1M tokens

### 前端设计约定
- 使用 `frontend-design` 技能控制 AI 味 UI
- shadcn/ui 组件 + Tailwind v4 自定义主题
- 字体：Noto Sans SC（正文） + Playfair Display（标题 / display）
- 配色：暖米色底 #f5f3f0 + 深棕文字 #2c2c2c + 陶土色 accent #b76e4b
- 整体风格：editorial / warm minimal，噪声纹理背景，玻璃卡片
- 动画策略：hover 微抬 0.5px + 封面 scale 1.04，避免过度动效
- 圆角：卡片 1rem，按钮 0.75rem，所有边框使用低透明度细线
- 字体策略：正文 12-14px，标签 10px，状态指示用 1.5px 圆点
- 参考 `vercel-react-best-practices` 进行性能优化
